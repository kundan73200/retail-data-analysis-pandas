"Analyzed e-commerce sales data using Pandas to extract insights on revenue, customers, products, and payment trends."
# 📊 E-Commerce Sales Analysis

## 📌 About this project

In this project, I worked on a real-world type e-commerce dataset using Python (Pandas).
My main goal was to analyze the data and find useful insights about sales, customers, and business performance.

---

## 🛠️ Tools I used

* Python
* Pandas
* OpenPyXL

---

## 📊 What I did in this project

* Cleaned the dataset (handled missing values, removed duplicates)
* Converted date column into proper format
* Created new column (total_amount)
* Analyzed monthly sales
* Found top selling products
* Found top 10 customers by spending
* Calculated cancellation rate
* Analyzed payment methods
* Found repeat customers
* Checked city-wise sales performance

---

## 📈 What I found

* Some products generate most of the revenue
* Few customers contribute more to total sales
* One payment method is used more than others
* Cancellation rate is present and needs improvement
* Some cities perform better in terms of sales

---

## 📁 Output

I saved all results into an Excel file with multiple sheets like:

* Payment details
* Monthly sales
* Top products
* Customer analysis
